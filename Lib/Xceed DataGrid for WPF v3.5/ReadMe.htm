Introduction

Thank you for installing Xceed DataGrid for WPF. The Professional Edition of Xceed DataGrid for WPF is not only the most mature and therefore rock-solid, time-tested grid control for Microsoft's Windows Presentation Foundation on the market. It's also the most powerful, with more features than any other offering and a flexible, extensible object model. Add that it's the fastest and that it provides unbeatable performance, handling millions of rows and thousands of columns while delivering an exceptional user experience, and it becomes easy to understand why it's the most-adopted WPF datagrid available: its tens of thousands of users know they have the best and regularly say so on one of the most active WPF forums around!

Xceed DataGrid for WPF is also available as a Standard Edition. This alternative to Microsoft's free datagrid lets developers create better apps faster, at an economical price.

Xceed DataGrid for WPF is available in cost-effective team and site licenses. Standard e-mail support is provided, but adding a Vanguard subscription gives you upgrades and priority support for an entire year. Full C# and XAML source code is available in the Blueprint Edition.

Both versions provide comprehensive documentation that fully integrates into Visual Studio, and C# and VB.NET sample applications.

What's new in version 3.5

Xceed DataGrid for WPF Professional Edition v3.5 features a beautiful new view called Tableflow™ View. This view offers fast animated smooth scrolling (with inertia) for impressive UI performance, (works with data virtualization), sticky group headers, sticky master-detail master row and headers, full-column animated drag & drop reordering. Also new is filtering in a virtualized collection view, and status indicator glyphs. Various minor bugs have been addressed. Version 3.5 of Xceed DataGrid for WPF Standard Edition features various minor bug fixes.

Installing

The installer copies all the control's files to the appropriate locations, but you still need to take additional steps to make the component available from your particular development environment. To do this, please consult the Licensing (Unlocking) the Component topic in the documentation and follow all the steps.

Licensing

To use Xceed DataGrid for WPF, the LicenseKey property should be set to either a registered license key that you have purchased or a trial key. Trial keys for both the Professional Edition and the Standard Edition can be obtained from the Resource Center application, which can be run from the Start Menu under All Programs -> Xceed DataGrid for WPF vX.X -> Xceed DataGrid for WPF vX.X Resource Center. 

The component remains fully functional for the trial period, which begins at the time of installation and lasts for 45 days. Once the trial period has ended, a registered license key is needed to continue using the product. If you attempt to use a component after the trial period without a registered license key, an exception will be thrown and you will not be able to use it.

Xceed DataGrid for WPF may be used without a license key for the 45-day trial period; by default, the Professional Edition of the product is active when no license key is explicitly set during the trial period. This is not however recommended: you should always license the component explicitly to avoid licensing issues. If, during the trial period, you wish to test your application on another computer, you must assign a key to the LicenseKey property as described above.

If you have purchased a registered license key, assigning it to the LicenseKey property will remove the trial limitations. The LicenseKey property must be set with a valid license key before any method of your Xceed component is called. If you use an invalid or expired license key, fail to license the control with a registered license key after the trial period, or license it in the wrong place in your code, an exception will be thrown at runtime.

For more information on how the 45-day trial works or how to license the component once you purchase, refer to the Licensing (Unlocking) the Component topic in the documentation.
